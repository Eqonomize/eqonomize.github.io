Eqonomize! downloads


(!) Note that even though most planned features are in place and I myself find it highly stable and reliable in everyday use, Eqonomize! definitely needs more testing and no guarantees whatsoever are given :-o

With that said, you are welcome to download the latest version of Eqonomize! and report any bugs you find.

Eqonomize! version 1.4.2 (2019-06-24)

  • Source code
    Requirements: Qt 5 (preferably with Qt Charts module)
    Installation instructions
  • Windows binary
    Requirements: Visual C++ Redistributable

  • Linux binary (Ubuntu 16.0.4, 64-bit)
    Linux binary (Ubuntu 16.0.4, 32-bit)
    Unpack the single executable file and run without installation. It will likely run on linux systems other than Ubuntu 16.0.4. Users of later Ubuntu versions need to install libqt5printsupport5.

  • Linux AppImage binary
    Make the file executable and run from anywhere without installation.

  • Linux AppImage binary (git version)

  • An experimental binary snap package for Linux, with all dependencies included, is available at snapcraft.io/eqonomize-hk for those using a slow moving GNU/Linux distribution (install with the command "sudo snap install eqonomize-hk").
