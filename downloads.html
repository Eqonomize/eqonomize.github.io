Eqonomize! downloads
(!) Note that even though most planned features are in place and I personally find it highly stable and reliable in everyday use, Eqonomize! definitely needs more testing and no guarantees whatsoever are given. With that said, you are welcome to download the latest version of Eqonomize! and report any bugs you find.
Windows binaries:
  • 64-bit installer
  • 64-bit "portable" zip package
  • 32-bit installer
The 64-bit versions are recommended, but requires Windows 10 or newer. The portable file can be unpacked and run without further installation and no application data are read or written outside of the unpacked directory. Visual C++ Redistributable version 17 is required (download 64-bit or 32-bit version)

Linux binaries:
  • Self-contained binaries: 64-bit, 32-bit
    Unpack the single executable file and run without installation. It will likely run on linux systems other than Ubuntu 16.0.4. Users of later Ubuntu versions need to install libqt5printsupport5.

  • Linux AppImage binary
    Make the file executable and run from anywhere without installation.

  • A binary snap package, with all dependencies included, is available at snapcraft.io/eqonomize-hk (install with the command sudo snap install eqonomize-hk).

    • Some distributions provide packages (which may or may not be outdated) which you can install using the distribution's standard installers. Eqonomize! can be installed on Debian and Ubuntu using sudo apt install eqonomize, and on Arch Linux from AUR.
